API Documentation

Build powerful automations with our RESTful API

Getting Started

The Agents Cloud API is designed for developers and AI agents who want to automate domain registration, DNS management, and hosting deployment. All endpoints return JSON and support both authentication via API keys and crypto payments.

Base URL

https://api.agentscloud.dev/v1

Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Core Endpoints

Check Domain Availability

POST /domains/check
{ "domain": "example.com" } // Response { "success": true, "available": true, "price_usd": 10.81, "tld": "com" }

Create Domain Order

POST /orders/create-fast
{ "domain": "example.com", "crypto_currency": "usdc", "customer_email": "user@example.com" } // Response { "success": true, "order": { "order_id": "ord_123abc", "domain": "example.com", "price_usd": 10.81, "price_crypto": 10.85, "crypto_currency": "usdc", "wallet_address": "0x...", "expires_at": 1710547200 } }

Check Order Status

GET /orders/:id/status
// Response { "success": true, "status": "completed", "domain": "example.com", "tx_hash": "0x..." }

List Your Domains

GET /domains/list
// Response { "success": true, "domains": [ { "domain": "example.com", "status": "active", "expires_at": "2027-03-15", "auto_renew": true } ] }

Update DNS Records

POST /dns/create
{ "domain": "example.com", "type": "A", "name": "www", "content": "1.2.3.4", "ttl": 3600 }

Rate Limits

Pro Tier: 1,000 requests/hour

SDKs & Libraries

We provide official SDKs for popular languages:

  • Node.js: npm install @agentscloud/sdk
  • Python: pip install agentscloud
  • Go: go get github.com/agentscloud/go-sdk
  • Rust: cargo add agentscloud

Webhooks

Subscribe to real-time events like domain registration, payment confirmation, and DNS updates. Configure webhooks in your dashboard.
